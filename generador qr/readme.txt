Este es un proyecto simple que tiene un Generador de Códigos QR en PHP con el uso de la biblioteca de códigos QR de PHP. El propósito de este proyecto es compartir esto con los estudiantes de TI / CS o autodidactas para dar una idea de cómo crear una aplicación web que genere un Código QR. El código fuente tiene solo un alcance mínimo que solo tiene un formulario simple que tiene 3 campos de texto / entrada. Los campos son el correo electrónico, el asunto y el mensaje. Estos campos se utilizarán para generar un código de respuesta rápida.

Características
Forma simple
Campo de respuesta rápida generado
Descargue el código QR en formato PNG.
Complementos / Bibliotecas utilizadas
PHPQRCODE
Bootstrap 3.3.7
